# stringkata-php

Quick runthrough of the string calculator kata at
http://osherove.com/tdd-kata-1.  Mostly a feel-good demo for me to get
something up and running on Travis CI :)

## Prerequisites

* PHP >= 5.3
* Composer (http://getcomposer.org/download)

## Running tests

    $ make test-install
    $ make test
